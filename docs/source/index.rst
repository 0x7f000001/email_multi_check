email-multi-check documentation
===============================

Email-multi-check is a check email toolkit for Python.
It can be used for queries, syntax check, quota check,
MX DNS testing, and many otherthings. Almost all types
are supported - Syntax, RCPT, VRFY, EXPN, WEB(requests).
